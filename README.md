# Chat
Um sistema de chat para comunicação entre usuários cadastrados

## Rodando o projeto 
 Para executar o projeto basta estar dentro do diretório ```/src``` e executar ```npm start ```

## Acessando o projeto
O projeto estará disponível em ``` http://localhost:4000/```
